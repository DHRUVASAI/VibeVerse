# Screenshots Guide

## How to Add Screenshots

1. **Take screenshots** of your VibeVerse app:
   - Hero Section & Splash Screen
   - Mood Selection Interface
   - Movie Recommendations
   - Music Player
   - Vibe Map (Emotional Galaxy)
   - Vibe Journal

2. **Save images** with these exact names:
   - `hero-section.png` (or `.jpg`)
   - `mood-selection.png`
   - `movie-recommendations.png`
   - `music-player.png`
   - `vibe-map.png`
   - `vibe-journal.png`

3. **Recommended settings**:
   - **Format**: PNG or JPG
   - **Size**: 1920x1080 or 1280x720 (16:9 aspect ratio)
   - **File size**: Under 1MB per image (optimize if needed)

4. **Upload to this folder** (`docs/images/`)

5. **Commit and push**:
   ```bash
   git add docs/images/*.png
   git commit -m "ðŸ“¸ Add app screenshots"
   git push origin master
   ```

## Alternative: Using GitHub Issues for Images

If you want to add images quickly:
1. Create a GitHub Issue
2. Drag and drop images into the issue
3. Copy the image URLs GitHub generates
4. Update README.md with those URLs

## Image Optimization Tools

- [TinyPNG](https://tinypng.com/) - Compress PNG/JPG images
- [Squoosh](https://squoosh.app/) - Google's image optimizer
- [ImageOptim](https://imageoptim.com/) - Mac image optimizer
